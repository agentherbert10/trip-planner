# Hawaii Trip Planner

A simple, single-page web app to organize your Hawaii trip from Feb 27 - Mar 7.

## Features

- **Flights** — Pre-loaded departure and return flight info
- **Itinerary** — Add activities, events, and things to do with dates and times
- **Packing List** — Interactive checklist to track what you've packed
- **Restaurant Map** — Map of Hawaii with restaurants you want to visit, mark them as visited when you go

## How to Run

1. Open `index.html` in a web browser (double-click or drag into Safari/Chrome)
2. All data saves automatically to your browser's local storage

## How to Use

### Itinerary
- Add an activity with a date and optional time
- Activities are sorted by date automatically

### Packing List
- Add items one at a time
- Check the box when packed
- Delete items you don't need

### Restaurants
- Add restaurant name and location
- (Optional) Add latitude/longitude to place it on the map
- Mark as "Visited" when you go
- Click on map markers to see restaurant details

## Data Storage

Everything is saved in your browser's local storage — nothing is sent to a server. If you clear your browser data, you'll lose your list. To be safe, back it up before clearing cache.
